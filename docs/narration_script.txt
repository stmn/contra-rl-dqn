Welcome to the Contra RL live training dashboard.

On the left is the live game preview. The agent controls the player in real time. In the bottom right corner is the agent's actual view — a grayscale, downscaled image with sprite overlays. Enemy positions and bullets are read from the game's RAM and drawn as shapes: rectangles for soldiers, circles for bullets, triangles for turrets, diamonds for bonuses. You can click the preview to swap the main and agent views.

In the header you can see the current level name, episode count, training time, how full the replay buffer is, and RAM usage.

Below the preview are the control buttons. First, three icon-only buttons: pause and resume, loop toggle which controls whether the agent auto-restarts after game over, and reset which forces a new episode.

Next group: Save Model creates a manual checkpoint of the neural network weights. Replay opens a recording of the best run for the current level.

Last group: Save State captures the current game moment. From that point on, the agent will respawn here after every death instead of restarting the level. This is called practice mode — useful for training on difficult sections like boss fights. A blue marker appears on the progress bar showing where practice begins. Clear State exits practice mode and returns to normal training.

Now the tabs on the right side.

The first tab is Overview. It shows total episodes played, total timesteps processed, training speed in steps per second, what percentage of episodes hit the time limit, how many auto-rollbacks happened, and when the last auto-save occurred.

The second tab is Live, which has five sub-tabs for real-time information about the current episode.

The Rewards sub-tab shows a color-coded breakdown: green for scroll reward, orange for kill reward, blue for turret and boss hits, purple for weapon upgrades, red for death penalties.

The Events sub-tab shows a timestamped feed of everything that happened in this run. Kills show the enemy type, turret hits show HP changes, deaths and weapon pickups are logged. Each line has a copy button on the left.

The Agent View sub-tab shows exactly what the agent knows right now. Player position and current weapon. A list of visible enemies with their type name, position, distance, and HP for turrets. A list of visible bullets with position and distance. Unknown enemy types appear in orange. Hovering over any coordinates draws a yellow crosshair on the game preview, so you can verify the position matches what you see.

The Features sub-tab shows the raw twenty-eight numbers the agent receives each frame — player position, weapon type, nearest enemies with velocities and HP, nearest bullets, invincibility timer, and level progress. Coordinate features also have hoverable crosshairs.

The Actions sub-tab shows how often the agent picks each of the sixteen button combinations, displayed as percentage bars.

The third tab is Leaderboard — the best runs for the current level, ranked by total reward.

The fourth tab is Levels. Each of the eight Contra levels has its own neural network model and separate statistics. You can switch levels by clicking the buttons here, or using the keyboard shortcut Command or Control plus one through eight. The level names come from the original ROM: Jungle, Base 1, Waterfall, Base 2, Snow Field, Energy Zone, Hangar, and Alien's Lair.

The fifth tab is Config, listing all training parameters and feature flags. Hover over any parameter to see a tooltip explaining what it does.

Below the tabs is the Level Progress bar. The green fill shows how far the agent is in the current run. Red marks show where the agent has died in previous runs, forming a death heatmap. The yellow PB line marks the personal best — the furthest point ever reached on this level.

At the bottom is the Reward History chart. The red area shows raw reward for each episode. The yellow line is the rolling average — when it trends upward, the agent is improving. The purple line shows the percentage of episodes where the agent reached the level boss.

Each dataset can be toggled on or off using the checkboxes in the legend. You can switch between chart and table view. Two sliders control the data range and the averaging window size. Hovering over the chart shows a vertical crosshair with values for all visible datasets.

You can also use keyboard shortcuts anywhere on the dashboard. Space to pause and resume. Right arrow to advance one step while paused. Command or Control plus a number to switch levels.
